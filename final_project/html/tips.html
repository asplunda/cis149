Tips & Tricks

The biggest tip I have for a new player in any MMO is to learn to set up key bindings. You will thank yourself later if you learn to play using mostly the keyboard rather than clicking everything in the game like crazy. Key bindings are your friend. Appreciate your friend.

Back to the social aspect. This is an MMO that you are playing. If you treat it like a single player game, then what's the point? Go out there and make friends. You will enjoy yourself a whole lot more!
